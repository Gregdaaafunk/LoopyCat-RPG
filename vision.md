# LoopyCat RPG

Internal title: Loopy TV RPG

## Phase 1 Focus

Detect toy, lock the canonical marker, keep a stable boss attached to toy space, run a real battle, and save a video where the boss remains visible.

## Phase 2 Focus

Make toy battles feel like an arcade game with random bosses, dramatic presentation, portal summoning, HP phases, combat effects, KO, victory, and placeholder loot.

The toy should feel like it opened a magical arcade portal and summoned a boss from the marker.

## Phase 3 Focus

Let the user create a cat profile, win rewards from battles, equip collected items in profile slots, and start building their own cat hero.

No AR fitting or live cat tracking yet.

## Phase 4 Focus

Turn the cat profile into a Cat Hero using multi-angle photos, local template animation, victory hero sequences, and equipment-driven appearance growth.

Photo cat -> create hero -> hero fights bosses -> hero evolves -> loot changes appearance.

The battle intro should make the player feel: "My cat entered battle."

## Phase 5 Focus

Turn boss victory into a real reward loop: loot drops, items are saved, equipment can be equipped, and the Cat Hero visibly progresses through placeholders.

Player defeats boss -> gets loot -> upgrades cat hero -> becomes stronger.

## Phase 6 Focus

Let the player see equipped rewards on the real cat through Photo Fit and Live Camera Fit, using manual placement now and future tracking anchors later.

Player wins item -> equips item -> sees item on real cat.

## Phase 7 Focus

Add progression, evolution stages, cinematic major victories, and Hall of Fame so the Cat Hero becomes a legend over time.

Cat becomes hero. Hero becomes legend.

## Phase 8 Focus

Create Loopy World: local worlds, seasonal content, boss rotations, achievements, and collections that give the player a reason to return every day.

Player returns every day. New boss. New loot. New world.

## Phase 9 Focus

Let the player locally save and show off victories through Cat showcase, share cards, screenshots, posters, profile stats, and galleries.

Player wants to show: "My cat defeated this boss."

## Phase 10-14 Focus

Connect physical toys to digital unlocks, build the Monster Book, expand equipment depth, create the Cat Kingdom, and push the Cat Hero into the Legend layer.

Physical toy -> AR battle -> Boss fight -> Loot -> Cat Hero -> Equipment -> Inventory -> AR fitting -> Collections -> Monster Book -> Cat Kingdom -> Legends -> Loopy Universe
