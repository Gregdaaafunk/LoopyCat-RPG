# Phase 10-14 Roadmap

Project title: LoopyCat RPG

Internal title: Loopy TV RPG

## Phase 10: Physical Toy Unlocks

Goal: Real toy unlocks digital content.

Build:

- Toy ID system
- QR support
- Physical unlock system
- Exclusive loot
- Exclusive bosses
- Hidden rewards
- Serial support

Toy classes:

- Classic
- AR Edition
- Elite Edition
- Event Edition
- Collector Edition

Flow:

1. Toy scanned.
2. Unlock content.
3. Unlock bosses.
4. Unlock cosmetics.
